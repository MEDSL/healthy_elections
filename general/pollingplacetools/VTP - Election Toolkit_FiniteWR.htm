Socially Distanced Polling Place Queue Model

COVID-19 Polling Place Queue Length Model

In the fall 2020 elections, polling places may institute occupancy limits far below their pre-COVID levels, creating a maximum line length inside the polling place. If only a few voters at a time may wait inside each polling place, lines may develop outside, and people may decide to go home without voting.

Input Data Description

Polling Place Characteristics

  1. Building Capacity: The total number of people allowed in the polling place at any time
  2. Election Staff: The maximum number of non-voters expected to be in the building at any time
  3. Voter Stations: The total number of different places a voter could be while not in line, including bottleneck stations
  4. Bottleneck Service Statistics: The number of stations and average service time on the slowest step in the voting process (e.g. check in). For more information, see the Voting Technology Project's previous voter queue calulator

Voter Behavior Characteristics

  1. Arrival Rate: The average rate (voters/hour) that arrive at the polling place and consider joining the line
  2. Outside Line Entry Chance: The chance that an arriving voter will join the unofficial line outside the polling place when the building is at capacity

COVID-19 uncertainty extends to estimates of voter behavior like the 'outside line entry chance'--factors like the weather on the day of have large impacts. Model Assumptions

This tool uses queueing theory to model a polling site which limits the number of voters that may be present at any one time.

The tool evaluates a classic queuing model, known as an “M/M/k system with a finite waiting room.” For modeling a polling site, we assume that voters arrive at a random rate, and join a single queue that is being served by a set of parallel stations or servers. In a polling site, the parallel stations could represent the check in operation, or the set of voting booths or voting machines. The choice between check in stations versus voting machines or booth should depend on whichever is regarded to be the bottleneck for the site.

In addition for this model, we assume that the polling site has a maximum occupancy, and will limit the number of voters that can be in the site at any time. We capture this in the model by imposing a limit on the number of voters that can be in line waiting at any point in time.

The model assumes that if the line is full, then any arriving voters are blocked and cannot enter the polling site. We assume that a blocked voter will leave and go home with some probability, which is an input we call the Balking probability. Otherwise, we assume that the blocked voter will wait outside the polling site, presumably in an outside queue, until he or she can enter the site.

In addition to using the tool on this web page, you can download an Excel spreadsheet that will perform the same calculations. One advantage of the spreadsheet is that it is easier to analyze multiple precincts at one time.
